# Предложения к следующим версиям

Цель из issue 1: ресурсная стратегия без внутриигровых валют, где люди с
потребностями строят поселение на планете, развивают технологии и показывают
преимущество ресурсной организации над соседними капиталистическими городами.

## Нерешённые шаги

- Сделать карту управляемой: выбор стартового тайла, просмотр биома, рек,
  высот, ресурсов и климата.
- Добавить строительство: постановка зданий на карту, пауза зданий,
  очереди работ и занятость людей переноской ресурсов.
- Разделить людей на жителей с состояниями: возраст, здоровье, травмы,
  потребности в еде, воде, тепле, жилье, электричестве и медицине.
- Расширить цепочки ресурсов: добыча, хранение, переработка, износ
  инструментов, отходы и повторное использование.
- Связать технологии с реальными производственными действиями, а не только
  с наличием ресурсов на складе.
- Добавить роботов как позднюю ветку: прочность, обслуживание, детали,
  зависимость от электроэнергии и производственных цепочек.
- Добавить соседние города с денежной экономикой для сравнения скорости
  развития, устойчивости и качества жизни.
- Подготовить интерфейс для смартфона: карта, склад, жители, технологии,
  скорость времени и журнал событий.
- Исследовать серверный режим: районы или города разных игроков, общая
  планета, ограничения паузы и расчёт симуляции на обычном ПК.
